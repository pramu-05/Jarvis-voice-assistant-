**Jarvis** is a Python-based local voice assistant that can listen, speak, perform web searches, play media, tell jokes, open apps and more.

## Features
- Speech-to-text (listen to your voice commands)
- Text-to-speech (Jarvis replies aloud)
- Play YouTube videos, search Wikipedia, tell jokes, open websites
- Run system commands and launch applications
- Easily extendable with new skills
